### pull 땡겨 오는 것 (푸쉬 전 풀)

```
git pull origin master
```

지정해놓은 repo에서 끌어온다.



##### push 자료를 올리는 것

```
git status
```



```
git add .
```

모든 파일들을 트래핑 해달라는 뜻

```
git commit -m "message"
```

업로드 했을 때 나에게 알려주는 commit 다른 팀원들에게 명령어다.



```
git push origin +master
```
